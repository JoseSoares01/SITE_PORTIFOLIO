Análise de Acidentes de Trânsito

Um estudo abrangente sobre os padrões de acidentes de carro para identificar fatores de risco e otimizar a segurança viária.

Dashboard de Acidentes (Power BI Embedded)

Introdução à Análise

O objetivo deste projeto foi explorar um dataset de acidentes de trânsito para descobrir insights que poderiam informar políticas de segurança pública. A análise focou em encontrar correlações entre condições climáticas, horário, tipo de via e gravidade dos acidentes.

Como mostra o gráfico acima, a maior concentração de acidentes ocorre nos horários de pico, especialmente entre 17h e 19h.

Principais Insights

Relação com o Clima

Analisamos que dias chuvosos aumentam a probabilidade de acidentes em 30%, e a gravidade também tende a ser maior.

Tipos de Via

As autoestradas, apesar de terem menos acidentes em geral, apresentam uma maior taxa de fatalidade devido à alta velocidade.

Localização de Risco

Identificamos os 10 cruzamentos mais perigosos com base na frequência e gravidade dos incidentes, sugerindo uma priorização para melhorias de infraestrutura.

Metodologia e Conclusão

Utilizei Python para a limpeza e transformação dos dados (bibliotecas Pandas e NumPy), e Power BI para a visualização e criação do dashboard interativo. A análise foi concluída com um relatório detalhado e recomendações para autoridades de trânsito.

Este projeto demonstra como a análise de dados pode ser uma ferramenta poderosa para a segurança pública, transformando dados brutos em decisões que salvam vidas.
